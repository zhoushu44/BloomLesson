# 幼芽备课台

一款面向幼儿园老师的教案生成器，帮助老师围绕 3-6 岁幼儿的发展特点，快速生成更贴合教学场景的活动教案。项目支持快捷模板、项目建议刷新、教案生成、历史保存、复制与导出，适合日常备课、活动设计和教研参考。

## 核心功能

- **一键生成，快速出稿**：输入年龄段、活动类型和教学目标，即可快速生成一份结构完整、内容清晰的幼儿园教案，帮助老师把更多时间留给课堂设计与师幼互动。
- **模板灵感，开箱即用**：内置多种常用教学模板，覆盖春天观察、生活习惯养成、绘本延伸、户外探索等高频场景，点一下就能进入备课状态。
- **项目建议，随时刷新**：当你临时想换一个活动方向，或需要更多灵感时，刷新项目建议即可获得更贴合当前场景的新方案，让备课不再卡壳。
- **内容完整，直接可用**：生成结果包含活动目标、活动准备、活动过程、注意事项和活动延伸，结构清楚、层次分明，拿来就能继续整理成正式教案。
- **历史记录，随时回看**：系统会自动保存你生成过的教案，方便后续重新打开、继续修改或删除旧记录，让每一次备课都更高效。
- **复制导出，灵活分享**：支持一键复制和导出文本，方便快速粘贴到 Word、飞书、钉钉等工具中，轻松完成二次编辑和分享。
- **本地部署，轻量省心**：无需复杂账号体系，下载安装到本地即可使用，也可以通过 Docker 快速部署到服务器，适合个人、教研组和小团队使用。
- **操作简单，上手即会**：页面流程非常直观，按照“选模板 → 填需求 → 生成 → 保存/导出”的顺序操作即可，新手老师也能很快熟悉。

## 本地运行

```bash
npm install
npm start
```

启动后访问：

```text
http://localhost:5089
```

## Docker 运行

```bash
docker build -t kindergarten-lesson-generator .
docker run -p 5089:5089 kindergarten-lesson-generator
```

## 自动发布

仓库已配置 GitHub Actions。向 `main` 或 `master` 分支 push 后，会自动构建并推送 Docker 镜像到 Docker Hub。

需要在 GitHub Secrets 中配置：

- `DOCKER_HUB_USERNAME`
- `DOCKER_HUB_TOKEN`

## 项目结构

- `index.html`：页面结构
- `app.js`：前端交互与生成流程
- `server.js`：Express 服务
- `style.css`：页面样式
- `Dockerfile`：容器化配置
- `.github/workflows/docker-publish.yml`：镜像自动发布流程
